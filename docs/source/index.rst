ROMANS
======

"ROMANS" is short for "Reduced Order Modeling for Accelerating Numerical Simulations".
This project consists of Python utitlies that facilitate the analysis and acceleration of 
numerical simulations using proxy machine learning models.  The code itself is also 
referenced using the term "romans", e.g. ``import romans``.
